## How to install
* Check dependencies before installing
```bash
git clone https://github.com/Yarhi/Murphy_Studio.git  
cd Murphy_Studio/  
cd src/  
javac */*.java
```

## How to run
```bash
java -cp . application.main
```

## Dependencies
Ubuntu :
```bash
sudo apt install openjdk-8-jdk openjdk-8-jre openjfx
```
